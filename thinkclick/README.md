# ThinkClick

A structured reflection tool by Pattenden Peak Performance.

## Deploy to Vercel

1. Upload this folder to a GitHub repository
2. Go to vercel.com and connect the repository
3. Add environment variable: `ANTHROPIC_API_KEY` = your Anthropic API key
4. Deploy

## Local development

```bash
npm install
# Create .env.local with: ANTHROPIC_API_KEY=your_key_here
npm run dev
```
